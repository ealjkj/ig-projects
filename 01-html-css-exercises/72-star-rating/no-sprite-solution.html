Star rating system (Question 72).

Create a star-rating system with hover using only HTML/CSS. No javascript.

  1. Use elements that are semantic compliant when possible.
  2. Assume you have a sprite with 5 stars selected and 5 stars unselected.
  3. The selection should persist

Solution

Description: We can nest the stars. Each nest is made out of one star and one star-nest, except the last nest which only contains a star.

When we hover a star-nest we turn all stars in the nest yellow (including childrens, granchildrens and so on). Also, every star that comes immediately after a nest will be gray.

Notes: Page is not responsive.
